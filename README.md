# RankedEngine.ai

**RankedEngine.ai** — это открытый инструмент для игроков Brawl Stars, помогающий принимать решения в драфтах соревновательного режима. Это не "читерский софт", а быстрый аналитический ассистент, который берет на себя рутину: анализ пиков, поиск контрпиков и сверку с актуальной метой[span_1](start_span)[span_1](end_span).

## Техническое устройство и безопасность
Мне важно, чтобы вы понимали: здесь нет "черного ящика". 

* **Маршрут данных:** Ваш телефон -> Официальный сервер Google API (Gemini 3.5 Flash)[span_2](start_span)[span_2](end_span). У этого приложения нет собственного сервера. Я, как разработчик, не имею доступа к вашим скриншотам или ключам[span_3](start_span)[span_3](end_span).
* **Захват экрана:** Используется стандартный `MediaProjection API`. После сжатия кадра, исходный объект `Image` принудительно закрывается (`image.close()`), не оставляя "хвостов" в памяти[span_4](start_span)[span_4](end_span).
* **Личные ключи:** Вы используете свой собственный ключ от Google AI Studio[span_5](start_span)[span_5](end_span). Вы единственный владелец доступа, и вы видите все запросы в своей консоли Google[span_6](start_span)[span_6](end_span).

## Как это работает
1. **Анализ:** При нажатии «АНАЛИЗ» приложение делает снимок экрана, который сжимается и отправляется в Google Gemini[span_7](start_span)[span_7](end_span).
2. **Веб-поиск:** ИИ гуглит свежие тир-листы, чтобы давать советы на основе текущего патча[span_8](start_span)[span_8](end_span).
3. **Персонализация:** Вы заполняете списки своих бойцов 11 уровня, и ИИ предлагает пики только из вашего "гаража[span_9](start_span)"[span_9](end_span).

## Важное уточнение
Иногда вы можете видеть **«Ошибка API (429)»**. Это не взлом, а ограничение тарифа Google на количество поисковых запросов. Просто подождите 60 секунд[span_10](start_span)[span_10](end_span).
