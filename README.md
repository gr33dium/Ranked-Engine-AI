# RankedEngine.ai

**RankedEngine.ai** — это открытый инструмент для игроков Brawl Stars, помогающий принимать решения в драфтах соревновательного режима. Это не "читерский софт", а быстрый аналитический ассистент, который берет на себя рутину: анализ пиков, поиск контрпиков и сверку с актуальной метой.

## Техническое устройство и безопасность
Мне важно, чтобы вы понимали: здесь нет "черного ящика". 

* **Маршрут данных:** Ваш телефон -> Официальный сервер Google API (Gemini 3.5 Flash). У этого приложения нет собственного сервера. Я, как разработчик, не имею доступа к вашим скриншотам или ключам.
* **Захват экрана:** Используется стандартный `MediaProjection API`. После сжатия кадра, исходный объект `Image` принудительно закрывается (`image.close()`), не оставляя "хвостов" в памяти.
* **Личные ключи:** Вы используете свой собственный ключ от Google AI Studio. Вы единственный владелец доступа, и вы видите все запросы в своей консоли Google.

## Как это работает
1. **Анализ:** При нажатии «АНАЛИЗ» приложение делает снимок экрана, который сжимается и отправляется в Google Gemini.
2. **Веб-поиск:** ИИ гуглит свежие тир-листы, чтобы давать советы на основе текущего патча.
3. **Персонализация:** Вы заполняете списки своих бойцов 11 уровня, и ИИ предлагает пики только из вашего "гаража".

## Важное уточнение
Иногда вы можете видеть **«Ошибка API (429)»**. Это не взлом, а ограничение тарифа Google на количество поисковых запросов. Просто подождите 60 секунд.
